# Browse Species

The **Browse Species** page (`/browse`) allows users to search, filter, and discover different plant species within the LeafyLedger ecosystem.

## Overview
- **Path**: `/browse`
- **Purpose**: Provide a searchable database for users to explore plant species and learn more about their care requirements.

## Features
- **Search Functionality**: Users can search for plants by common name, scientific name, or keywords.
- **Species Details**: Displays images, descriptions, and basic care information for each plant.
- **Save Integration**: Users can bookmark or "save" specific species directly from the browse page to add them to their personal collection.

## User Flow
1. Navigate to "Browse Species" from the dashboard or navigation menu.
2. Use the search bar or visual grid to find plants of interest.
3. Click on a species to view detailed information.
4. Click the "Bookmark" icon to add the species to the Saved Species list.
